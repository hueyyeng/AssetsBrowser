Assets Browser v0.1.0
https://github.com/hueyyeng/AssetsBrowser

Last updated: 12 November 2017

Assets Browser is written in Python using PyQt5 for the GUI framework. This allow anyone to easily modify the code to fit it into their own pipeline structure.

Requirements (based on VFX Reference Platform CY2017):

  1. Python 2.7+ (tested on 2.7.14)
  2. PyQt 5.6 (tested on 5.7.1 on Win and 5.9.1 on macOS/Linux)

Usage

  1. Browse existing assets in your chosen project.
  2. Create new asset with a dialog options that are tailored to the project specs.
  3. Open selected file/directory path in your OS file manager.
